git init . 		-> Cria um repo git na pasta atual
git status 		-> Mostra o status do trabalho (de cada arquivo)
git add nome_arquivo    -> Adiciona arquivo para ser comitado
git commit -m "msg"     -> Faz commit do que stava para ser commitado
git log 		-> Mostra todo o historico de commit	

## FLUXO DE TRABALHO
1. criar o diretorio do projeto
2. iniciar o git dentro do diretorio criado ( git init . )
3. eidta ou cria arquivops
4. git status
5. git add <nome arquivo>
6. git status
7. git commit -m "Mensagem estranha"
8. git status
9. volta pro passo 3
